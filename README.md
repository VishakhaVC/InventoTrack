InventoTrack is a role-based inventory management and eCommerce web application designed to manage electronic products, stock, orders, and users efficiently. The system is built with three main modules: Admin, Manager, and User, each having specific responsibilities and access levels. The project focuses on maintaining accurate inventory records, smooth order processing, and a user-friendly shopping experience.

The platform allows administrators to control the entire system, including managing admins, managers, users, categories, products, stock, orders, and feedback. Managers handle sales-related operations and reports, while users can register, log in, browse products, add items to the cart, place orders, and manage their profiles. The system also includes features like email-based password recovery, OTP verification, low-stock alerts, and detailed reports.

InventoTrack is developed using PHP and MySQL for the backend and HTML, CSS, JavaScript, and Bootstrap for the frontend, running on an Apache server. It demonstrates real-world implementation of CRUD operations, authentication, authorization, role-based access control, and inventory tracking.

This project is suitable for learning and demonstrating full-stack web development concepts, especially in inventory management and eCommerce workflows.

~ Features
  - Admin Panel
      Admin login with validation
      Manage admins, managers, and users
      Manage categories and products (Add, Edit, Delete)
      Manage stock and orders
      View feedback
      Low stock email alert system
      Dashboard with stock and sales overview
      Generate product and order reports

  - Manager Panel
      Manager login
      Manage sales
      View reports
      Dashboard overview
      Secure logout

  - User Panel
      User registration and login
      Forgot password with email & OTP
      Browse categories and products
      View product details
      Add to cart & checkout
      Place orders and view order history
      Manage user profile and account details
